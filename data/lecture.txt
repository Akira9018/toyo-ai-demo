【第189回】
1. 棘下筋の使い方と診断
棘下筋を使用する際は、「水毒」や「瘀血」の症状に応じてC3を判定し、同側の棘下筋を考慮する 。

また、対側の肩に手を置き、肘と胸郭の間隔を調べることで、肩甲下筋との拮抗作用を利用した棘下筋の使い方もある 。

棘下筋の使用が必要かどうかの判定には、上下の動揺を検査することが有効 。

2. 円背と縦・横テープの貼付
円背の人には、腰・背部の局所に対して縦・横のテープを「引き上げて貼付」する方法が有効である 。

このテープ貼付方法を併用することで、円背の人が大幅に改善したという事例が報告されている 。


3. 足の趾（ゆび）と神経支配
足の第1趾の処置は、ほとんどの人に効果があるが、足が上がらない人や膝を上げて歩く人には第2・3趾、第4・5趾の処置がより有効である 。

神経支配は足底と足背で異なり、皮膚分節でみると全く違うが、神経の出てくる場所は同じである 。


第1趾への処置は、重心を後ろに下げる効果があり、これによってしびれが取れることがある 。重心が前にかかるとしびれが発生するため、重心を後ろに戻すことが重要である 。


足の5本の趾は重心を取る上で重要であり、5本指ソックスを履くことは悪影響を及ぼす可能性がある 。

4. 上焦・中焦とスケール値
「上焦」は伸筋、「中焦」は屈筋と単純に理解するべきではない 。

スケール値（0＞0）が大きい場合、右側に伸筋障害、左側に屈筋障害があるとされる 。

スケール値（0＜0）が大きい場合、左側に伸筋障害、右側に屈筋障害があるとされる 。

関節が近づきすぎるのは屈筋に異常があり、関節が曲がらないのは伸筋障害があると考えられる 。


スケール値が0.9や0.7の場合、冷えを意味するが、スケール値の前後の大小とは別の概念である 。

熱がある場合は左の肩鎖関節、冷えている場合は右の肩鎖関節が主となる 。

5. 症状の序列と思考法
症状には「前後＜左右＜上下＜前後、左右、上下（の複合）」という序列がある 。



この考え方に基づくと、左右の症状は前後よりも重く、上下の症状は左右よりも重いとされる 。

円背は、起立筋の筋力低下により背中の筋肉が緩み、体を丸めてしまうことで発生する 。


体内の陰圧の変化が様々な症状を引き起こすという考え方が重要である 。


円背の治療では、頭頂筋、頸長筋、横隔膜の腱、大腰筋といった筋肉の緊張を考慮する 。これらの筋肉が緊張すると、C2・C3や横隔膜、大腰筋の始まる部分に痛みが出ることがある 。

6. 棘上筋と肩甲骨
棘上筋の主な働きは、骨頭をわずかに引き寄せることであり、肩甲骨を引っ張るだけである 。

悪い状態の人は、肩甲骨を支えられなくなっており、これが痛みの原因と考えられている 。


腕を前に出すと、肩の重みが肋骨にかかってしまい、体を伸ばすことができなくなる 。

7. 内関（ツボ）の働き
内関を押さえると、意識が回復したり（目が開く）、首の側面の筋肉が緩んだりする 。



これは、呼吸に関連する筋肉や血管に関わる神経（舌咽神経、舌下神経など）の圧迫が軽減され、血液が脳に上がりやすくなるためである 。

内関は、急性の症状で首の側面の筋肉が締め付けられている場合に特に有効で、呼吸がしやすくなる 。


内関は陰圧に働きかけるツボと考えられている 。

【第190回 前編】
1. 膝の痛みと重心の捉え方
膝の痛みの原因は、右と左で異なるという考え方がある 。

右膝の痛みは主に大腿部に異常があり、左膝の痛みは下腿部に異常があるという 。

「軸・重心の捉え方」で重要なのは、足の5本の趾（ゆび）である 。

親指の処置が良く効くのは、歩行に問題がなく、単に痛みがある場合である 。

親指で地面を蹴れていない人や、膝を上げて歩く人は、第2・3趾や第4・5趾の処置が効果的である 。

足の趾の処置は、重心を後ろに下げる効果があり、これによって痺れが取れることがある 。

2. 筋肉と神経の関係
脊髄に影響を与える筋肉として、頭頂筋、頸長筋、横隔膜の腱、大腰筋が挙げられる 。

これらの筋肉が緊張すると、C2・C3や横隔膜の始まり、大腰筋の始まりの部分に痛みが生じる 。

肋間神経は、内臓の反射よりも大きな影響力を持つ非常に重要な神経である 。

腕の重みが肋間神経を刺激することが、肩の痛みの原因となる場合がある 。

3. 棘上筋と腱板断裂
棘上筋は、骨頭をわずかに引き寄せる働きしかなく、肩甲骨を支える働きが弱くなると痛みを引き起こす 。

腱板断裂は、筋の部分断裂か腱の部分断裂かによって予後が異なる 。

腱の部分断裂の場合は、断裂部分を近づけて数ヶ月かければ治癒する可能性があるが、筋の部分断裂は萎縮してしまうため治癒しない 。

棘上筋が断裂しても、棘下筋、小円筋、肩甲下筋が代償作用をするため、腕を上げることができる場合がある 。

高齢者の部分断裂は保存療法で済ませることも多いが、若年者や運動選手の場合は手術が選択されることがある 。

4. スパイラルテープ療法
スパイラルテープは、体液循環や情報伝達の異常を改善する目的で用いられる 。

股関節の関節唇と、中殿筋や大殿筋の処置は併用しない方が良い 。どちらかに絞った方が良いとされている 。

足底の縦・横アーチにテープを貼ると、上下方向の揺れが止まることがある 。

テープを貼る前に、大転子の位置や揺れを検査しておくことが重要である 。

5. 肘と手への処置
手の処置は、足と同様に第2・3指を中心に考え、決定した方を擦り、反対側も擦る 。

痛い方の肩が張っている場合、指を擦ることで肩の張りが軽減される 。

肘内障を起こした肘は重く感じることがある 。

手の重みを確認することで、肘の異常を発見できる場合がある 。

6. 内関のツボ
内関は、首の側面の筋肉を緩める効果がある 。

内関を押さえることで、呼吸がしやすくなったり、意識が回復したりすることがある 。

これは、首の周りの脳神経に働きかけ、脳への血流を改善するためと考えられている 。



【第190回 後編】
1. 「主従」の考え方
症状や病態を理解する上で、「主従」の考え方が重要である。

複数の病症がある場合、どちらが主な病の原因（主）で、どちらが補助的な病症（従）であるかを特定する。

例えば、「肝虚膀胱実」の場合、「肝虚」が主な病症である場合と、「膀胱実」が主な病症である場合とがある。

主従の判断によって、どの部分を重点的に治療するべきか、治療の優先順位が変わる。

「心虚胆実証」の場合、心が主な病症である方が症状が重いと考える。

2. 75難のパターンと治療法
75難には、3つのパターン（証）が存在する。

これらのパターンは、中焦、上焦、下焦のどれに問題があるかによって分類される。

右側が冷えている場合は腎虚、熱がある場合は膀胱実と判断する。

腎虚は右の中焦と下焦、膀胱実は左の中焦と下焦の問題と関連している。

施術の際は、これらのパターンを考慮し、中焦、上焦、下焦のどこに問題があるかを見極めることが重要である。

3. 下肢の検査と治療法
下肢の動揺を検査することが重要。

内転筋のこの位置を「チョンチョン」と叩くことで、回旋する力が元に戻り、動揺が改善されることがある。

この検査によって、股関節の開排制限が改善される事例も報告されている。

治療効果を持続させるために、縦にテープを貼ることが有効。

この治療法は、側弯症で上を向けない高齢者にも効果が見られる。

4. スケール値の判定
スケール値「0＞0」は、右の足で検査し、スケール値の大きい方を治療に利用する。

第2・3趾の痺れは神経支配が異なっており、胃経であっても異なる神経が支配している。

治療の際は、スケール値「0＞0」を基準に測定し、比較通流値の大きい方を治療する。

5. 後斜角筋と腰痛・臀部痛
後斜角筋は、首の後屈時の痛みだけでなく、腰椎部や臀部痛にも影響を与える。

後斜角筋を触診することで、良い治療効果が得られる場合がある。

6. その他
「横隔膜」は非常に重要なキーワード。

過去の先生方の研究から、さまざまな検査法や治療法が考案されている。
